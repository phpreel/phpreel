UrlRoutes::search($query)#

Returns URL to a search result based on a given query.

  • $query Required string, query to search for
/app/Helpers/Theme/UrlRoutes.php
use UrlRoutes;
...
UrlRoutes::search('new movies');

UrlRoutes::searchPost()#

POST: Query the database to search for a movie or series.

The parameters listed bellow must be sent to this POST route in order to make it work.

  • _token Required string, with every post request with Laravel you must add a hidden input with the value of csrf_token() or just use @csrf Blade directive
  • query Required string maximum 255 characters, word or group of words to search for in the database
/app/Helpers/Theme/UrlRoutes.php
use UrlRoutes;
...
UrlRoutes::searchPost();

UrlRoutes::loginPost()#

POST: Login a user.

The parameters listed bellow must be sent to this POST route in order to make it work.

  • _token Required string, with every post request with Laravel you must add a hidden input with the value of csrf_token() or just use @csrf Blade directive
  • email Required string, email address to which the account is registered
  • password Required string, password that corresponds to the email
  • remember Required string, checkbox if the user should be remembered when he comes back
/app/Helpers/Theme/UrlRoutes.php
use UrlRoutes;
...
UrlRoutes::loginPost();
