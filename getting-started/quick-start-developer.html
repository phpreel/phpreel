Quick start for developers

If you plan on using phpReel as a starting point for your next project then you should first fork this repository so you can have your own version of it. Right after that you can clone the forked repository and start the setup of the app.

At its core, phpReel is a Laravel app. This means that you have to follow the same steps as you would do with any Laravel app.

Install with Composer#

One of the easiest ways to install a new phpReel application is by using Composer. First make sure you have installed and configured PHP, MySQL (or you can install XAMPP to quickly install at once) and Composer on your computer. Next open up a terminal and run the following commands.

composer create-project phpreel/phpreel new-app
cd new-app
php artisan dev:database
php artisan migrate
php artisan dev:stripe
php artisan dev:init
php artisan serve

Next time you want to start your development server run php artisan serve in the root of your application.

Install with Bash script#

Here is a quick summary of what you will have to do:

  • Make sure you have previously installed on your computer: PHP, Composer, MySQL database, or other databases that work with Laravel, Laravel also provides an instalation guide which is fairly similar to what you will have to do here.
  • Clone your forked repository.
  • From here you have two options. You either run the initialization bash script (situated at the root of the project) from your terminal by calling bash init.sh and then follow the instructions from the terminal, or you continue following the next steps which ar equivalent to the manual installation of the application.

Remember that you should delete the bash script file called init.sh if you plan on releasing your app to production. The script is there just to help you with the development process!

Next time you want to start your development server run php artisan serve in the root of your application.

Install manually#

  • Get the contents of the .env.example file and save them to a .env file.
  • Open the console, cd to the root of the project, and run the following commands.
  • composer install
  • npm install
  • php artisan key:generate
  • Start configuring your database and Stripe settings by either editing the .env file directly or using Artisan commands in your terminal.
  • php artisan dev:database
  • php artisan migrate
  • php artisan dev:stripe
  • php artisan dev:init to create your admin account and add basic seeding data to the database
  • php artisan serve

Now your app is live and you can start working on it!

Good to know#

If you plan on developing content for phpReel you should change some variables for the .env folder. Set APP_ENV to local and APP_DEBUG to true to be able to see error messages right from the browser.

If you plan on publishing your app don't forget to set the APP_ENV to production and APP_DEBUG to false once you are done to hide sensitive information that might appear to the end user.
