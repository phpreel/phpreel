Try phpReel

Overview#

This guide will show you how to setup phpReel really fast and easy by executing just a few commands inside your terminal. This option is great if you install phpReel locally so you can try it out.

This is not intended for production use. If you are not interested in making your application public please follow the other guides.

Install locally#

One of the easiest ways to install a new phpReel application is by using Composer. First make sure you have installed and configured PHP, MySQL (or you can get XAMPP to quickly install both PHP and MySQL at once) and Composer on your computer. Next open up a terminal and run the following commands.

composer create-project phpreel/phpreel new-app
cd new-app
php artisan dev:database
php artisan migrate
php artisan dev:stripe
php artisan dev:init
php artisan serve

Next time you want to start your local server open a terminal at the root of your application and run php artisan serve.
